---

## 💡 关键概念

| 概念 | 说明 |
|------|------|
| VitePress | 静态网站生成框架，用 Markdown 写内容 |
| GitHub Pages | GitHub 提供的免费网站托管服务 |
| GitHub Actions | 自动化流程，push 后自动构建部署 |
| git proxy | 让 Git 走代理通道访问 GitHub |
| --dangerously-skip-permissions | Claude Code 全自动模式，无需每步确认 |
