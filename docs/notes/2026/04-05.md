## 📝 笔记内容

### 什么是 CLAUDE.md

CLAUDE.md 是放在项目根目录的特殊文件，用来指导 Claude Code 如何在这个代码库中工作。相当于给 AI 助手的一份「项目说明书」。

### 创建过程

```
运行 /init 命令
    ↓
Claude 分析项目结构
    ↓
生成精准的 CLAUDE.md
    ↓
推送到 GitHub
```

### CLAUDE.md 包含的关键信息

| 模块 | 说明 |
|------|------|
| 开发命令 | `docs:dev`、`docs:build`、`docs:preview` |
| 笔记组织 | `docs/notes/YYYY/MM-DD.md` 日期结构 |
| 侧边栏生成 | 自动扫描文件系统生成导航 |
| 部署配置 | GitHub Actions → GitHub Pages |

### 为什么需要它

- **项目结构特殊**：侧边栏是动态生成的，不是静态配置
- **中文内容**：`lang: 'zh-CN'` 设定
- **子目录部署**：`base: '/wiki/'` 设定
- **节省沟通成本**：以后不用每次都解释项目结构
