基础入门
01

Claude 界面快速上手

认识模型切换、联网功能、文件上传与连接器
入门
  • ①登录 claude.ai，界面左上角可切换模型：Sonnet（日常首选）与 Opus（复杂任务）。
  • ②右侧工具栏可开启 联网搜索，让 Claude 实时检索最新信息，效果远超直接用 Google。
  • ③上传文件支持：截图、PDF、GitHub 文件、Google Drive 文件，可直接拖入对话框。
  • ④在 连接器（Connectors） 里可接入 Gmail、Google Calendar、Zapier、MCP 服务等第三方 App。
  • ⑤扩展思考（Extended Thinking） 开关位于输入框底部，开启后回答质量更高，但消耗额度更多，建议复杂任务再用。
💡

讲师建议：日常工作把 Sonnet 当主力；需要深度推理（写复杂代码、分析长文档）时才切到 Opus 并开启扩展思考。

核心工作流
02

落地页自动生成

把任意网页信息喂给 Claude，几分钟得到可部署的 HTML 落地页
实操
  • ①打开目标网站，复制页面上的产品/服务信息（文字即可）。
  • ②粘贴进 Claude，附上指令：帮我用这些信息构建一个美观现代的落地页，目标 = 转化流量，CTA 按钮链接为 [你的链接]
  • ③Claude 自动在 Artifact（预览框） 里生成 HTML/CSS/JS，右侧实时预览效果。
  • ④不满意可继续指令微调：把配色改得更现代，添加客户评价模块，无需懂代码。
  • ⑤点击 Artifact 右上角 分享链接，立即上线可访问；或下载 HTML 上传到 Netlify / WordPress。
🚀

部署方案：① Artifact 内置分享链接（最快）② 下载后上传 Netlify 绑定自定义域名 ③ 复制 HTML 粘贴到 WordPress 页面编辑器

03

SEO 内容自动化流水线

视频字幕 → SEO 博文，5 篇文章一次生成，直接发布
实操
  • ①打开 YouTube 视频，点击 字幕 → 复制文字稿（Transcript）。
  • ②进入你的 SEO 写作 Project（见模块 4），粘贴字幕文字，发送给 Claude。
  • ③Claude 按照 Project 里预设的写作风格（语言简洁、关键词布局、标题格式），自动输出 SEO 博文，每句话单独成段，排版即发即用。
  • ④一次生成 5 篇不同角度的文章围绕同一关键词，分发到网站、LinkedIn、X (Twitter)、Reddit 等平台。
  • ⑤将文章链接粘贴回 LinkedIn 文章编辑器，含 CTA 链接，直接发布引流。
📊

效果数据：讲师从 9 月下旬开始用此流程，网站自然流量持续稳步上升。他认为 Google 不会惩罚 AI 内容，关键在于内容是否真正回答用户问题。

05

Twitter / X 内容训练系统

用数据驱动的方式，持续优化推文 Hook，提升触达量
进阶
  • ①建立 推文追踪表：记录每条推文的发布日期、链接、浏览量、使用的 Hook 类型。
  • ②定期分析数据，将 表现前 10% 的 Hook 整理为"继续做"清单，后 90% 的 Hook 列为"避免"清单。
  • ③把两份清单上传到 推文写作 Project 的 Files 中，Claude 自动学习"什么有效"。
  • ④每次创作时：粘贴视频字幕 → Project 自动参考历史数据 → 生成多个版本推文，选最可能病毒式传播的发布。
  • ⑤发布后更新追踪表，形成 数据反馈闭环，越用越精准。
📈

实际效果：讲师使用该系统后，单日触达从约 12 万提升至 22~32 万人。

高阶功能
06

Skills（可复用技能）

把任意工作流打包成一键命令，随时在对话中调用
专业
  • ①在 设置 → Capabilities → Skills 中创建技能，或在任意对话后说 将这个流程保存为技能，命名为 YouTube 标题生成器
  • ②Claude 自动根据对话内容提炼技能描述，存储到你的账号中。
  • ③任意新对话中，输入 使用 YouTube 标题技能，主题 = Claude AI 课程，Claude 立即按该技能格式生成内容。
  • ④技能可以导出分享，别人导入到自己的 Claude 账号即可使用，方便团队协作。
🎯

妙用举例：讲师内置了"顾问辩论技能"——输入一个商业主题，两位 AI 顾问（如 Alex Hormozi vs Russell Brunson 风格）从不同角度辩论，帮助快速梳理策略。

07

Claude Code + Cursor（终端编程代理）

在终端内用自然语言操控代码项目，无需懂编程
专业
  • ①Claude Code 是运行在命令行中的 AI 代理，通过 npm install -g @anthropic-ai/claude-code 安装。
  • ②与 Cursor（免费 AI IDE） 结合使用：在 Cursor 中打开项目，新建终端，输入 claude 启动代理。
  • ③用自然语言下达指令，例如：把这个待办事项应用改得更有趣，加入游戏化经验值系统，增加彩色动画，Claude Code 自动修改代码文件。
  • ④可直接输入 GitHub 仓库地址，让 Claude Code 帮你 克隆并配置环境，省去繁琐的手动操作。
  • ⑤优势：Cursor 内置 AI 额度有限，接入 Claude Code 后可获得 Sonnet 级别的更强输出，且不受 Cursor 额度限制。
⚠️

注意：Claude Code 执行操作前会询问是否确认，建议保持默认的"先询问"模式，避免误操作。

08

Computer Use（浏览器自动化代理）

Claude 控制你的浏览器，自动完成网页操作任务
实验性
  • ①安装 Claude for Chrome 扩展（目前 Beta 版），需 Pro 订阅。
  • ②两种模式：先询问再执行（推荐新手）/ 直接执行（高效但风险较大）。
  • ③示例操作：帮我写一篇关于日本 SEO 的文章，放到 Google Docs 里 → Claude 自动打开新标签页、创建 Docs、开始写作，全程无需手动操作。
  • ④可访问你已登录的所有网站（Gmail、LinkedIn、WordPress 等），实现跨平台自动化。
  • ⑤与 Zapier / MCP 结合，可间接连接 QuickBooks、Asana 等商业软件。
🔮

讲师评价："这个功能一年前还非常初级，现在已经能稳定完成复杂任务了，是我见过最接近'AI 员工'的工具。"

综合应用案例
+

直播内容全自动生成流程

从主题到六千字完整直播讲义，一条 Mega Prompt 搞定
彩蛋
  • ①在"直播笔记 Project"中预设 Mega Prompt：包含框架结构（Hook → Story → Offer）、内容要求、CTA 模板。
  • ②每次只需输入 主题 = 今天要讲的工具名，Claude 联网研究 20+ 信息源，自动生成完整讲义。
  • ③讲义包含：开场 Hook、自定义教学框架命名、案例故事、信念打破话术、直播推广帖、100 条相关提示词。
  • ④将讲义粘贴到 Google Docs（或用 Computer Use 自动填充），格式完美，约 30 页 / 6000 字，手动整理需一整天。
🏆

核心理念：把"输入你的时间"换成"输入正确的提示词"。Claude 帮你做调研、写框架，你只需专注在真正需要人类判断的部分。
